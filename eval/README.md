# Eval Workspace

This directory is reserved for evaluation-harness data and outputs.

- Harness run outputs: `eval/runs/<run_id>/`
- Full harness runner script: `eval/run_full_eval.sh`
- Keep source code under `src/eval/`
- Keep ad-hoc local QA files for harness runs here if needed

The default `run_ablation.py` output path is `eval/runs`.
